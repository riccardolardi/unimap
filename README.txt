unimap
A touch enabled application to explore Germany's Design universities

Using a Microsoft Surface touch table, TUIO & Processing, we created an application which allows the user to browse visually through all Design universities of Germany.

Made at Fachhochschule Potsdam
Summer Semester 2011
Interface Design course "Graspable Maps"
Lector: Till Nagel
Students: Julia Kontor, Sarah Baron, Paula Breithaupt, Riccardo Lardi

Full project documentation: http://incom.org/projekt/2014
